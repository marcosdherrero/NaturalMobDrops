# Natural Mob Drops

Fabric mod for **Minecraft Java 26.1.2** — natural spawn-egg and mob-head drops on player-credited kills, optional silk-touch spawner drops, and configurable Ender Dragon podium rewards.

## Requirements

- Java 25+
- [Fabric Loader](https://fabricmc.net/use/) 0.19.2+ for Minecraft 26.1.2
- [Fabric API](https://modrinth.com/mod/fabric-api)

## Build

```bash
./gradlew build
```

Output JAR: `build/libs/naturalmobdrops-1.0.0.jar`

## In-game

Command root: `/naturalmobdrops` (see `/naturalmobdrops help`).
